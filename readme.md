# Giga Pet JavaScript Project

A simple virtual pet game built with JavaScript and jQuery.

## Live Demo
🎮 **Play the game here:** 


## Features
- Feed your pet treats
- Play with your pet
- Exercise your pet
- Let your pet sleep to restore energy
- Watch the stats change in real-time
- See fun messages from your pet

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery

## How to Run Locally
1. Clone this repository
2. Open `index.html` in your web browser
3. Start taking care of your virtual pet!

## Project Requirements
This project was created for COMP484 to demonstrate:
- JavaScript objects
- Event handling with jQuery
- DOM manipulation
- Conditional logic
- Custom jQuery methods (.hover() and .slideDown())
